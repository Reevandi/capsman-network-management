# CAPsMAN-Based Wireless Network Management

## Project Status
✅ Completed (Team Project – Lab Simulation & Documentation)

## Overview
This repository documents a **team-based wireless network project**
focused on centralized access point management using
**MikroTik CAPsMAN (Controlled Access Point System Manager)**.

The project emphasizes structured network design,
collaborative planning, and technical documentation.

## Project Type
- Team Project
- Network Infrastructure Lab
- Academic / Technical Documentation

## Objectives
- Design a centralized wireless network architecture
- Apply CAPsMAN-based access point management
- Improve wireless consistency and scalability
- Produce clear technical documentation

## Technologies Used
- MikroTik RouterOS
- CAPsMAN
- WPA2/WPA3 Wireless Security
- Structured Network Documentation

## Team Contribution
My responsibilities in this team project included:
- Assisting in network topology design
- CAPsMAN configuration planning
- Wireless profile documentation
- Configuration review and validation

## Documentation
Detailed documentation is available in:
- `/topology` – Network diagrams and explanations
- `/configs` – Configuration planning documents
- `/docs` – Architecture and implementation details
- `/report` – Final project report

## Outcome
This project demonstrates the ability to collaborate
within a technical team to design and document
a centralized wireless network using MikroTik CAPsMAN.
